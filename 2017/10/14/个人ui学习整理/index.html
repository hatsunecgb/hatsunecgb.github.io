发表于 2017-10-14 | 分类于 学习

第一节 提高手绘

注意:

  1. 尺子画线
  2. 加立体线
  3. 加斜线阴影
  4. 加阴影
  5. 加阴影横线
  6. 减浅阴影

第二节 布尔运算与锚点

PS设置

  1. 编辑—首选项—工具—矢量工具变化与像素网格对齐√
  2. 形状：对齐边缘√

快捷键

  1. Ctrl+N 新建
  2. Ctrl+= 放大
  3. Ctrl+- 缩小

布尔运算

  • 合并形状：Ctrl+E
  • 打开：Ctrl+O
  • 减去顶层形状：选中什么减去什么
  • 与形状区域相交：减去不相交部分
  • 排除重叠形状：减去相交部分
  • 新建图层：准确新建形状

齿轮做法

多边形工具：调整边数，4的倍数（齿轮）
选中相隔的两个之后,Ctrl+T调整（齿轮）

第三节 微扁平图标绘制

  1. 布尔画出形状
  2. 根据图层样式调出想要的形状
  3. 注意细节
  4. 斜面和浮雕 描边 内阴影 内发光 渐变叠加（叠加才能设置蒙版） 投影

第四节 中国风创作思路及技法

步骤

  1. 确定某样事物
  2. 确定颜色
  3. 材质 质感 特点
  4. 环境因素

玉器制作需要图层

  1. 主图层：斜面和浮雕，内阴影，内发光 注：（增加凹效果需要浮雕）
  2. 两层阴影，近重远轻（有层次效果），投影
  3. 高光，增强高光反光效果
  4. 反光，增加边缘反光透明度
  5. 内阴影，增加暗部阴影
    顺序：34512

书本制作步骤

  1. 新建透视形状
  2. 复制一份，下拉
  3. 拉参考线
  4. 侧边上角布尔运算

  6. 钢笔锚点，书角弯曲细节

  8. 做书页复制最底层
  9. 侧边移到底层上
  10. 底层添加书页，添加锚点

第六节 重拟物创作思路及技法

步骤

  1. 设定元素
  2. 选择元素
  3. 可以改变设计外形
  4. 强化素材材质
  5. 环境因素—明暗对比
  6. 背景颜色与物品要相匹配

柠檬制作

  1. 画个形状，复制向上移动
  2. 果肉部分，一片果肉，一片材质，如果果肉不明显可对其进行锐化处理
  3. 可以对果肉进行描边，使其过度自然
  4. 皮内部分，皮内边缘材质，皮内暗部
  5. 皮外部分，皮外材质，柠檬皮暗部（两边），柠檬皮亮部（中间）（底部边缘），阴影

    第七节 银器设计思路及质感

步骤

  1. 利用布尔运算画出雏形，复制一层
  2. 缩进边缘，体现厚度
  3. 斜面浮雕，阴影
  4. 渐变阴影效果
  5. 暗部，细节
  6. 总结：参照实物，细节重要

    第八节 金币设计思路及技法

效果分析：

投影—渐变叠加（光照渐变）—描边（可做光照效果）—描边—（凹效果）—描边（来光照射效果）—内发光（机枪凹底阴影质感）—
外发光（加强凸底散光质感）—描边（加强边缘效果）

总结：

注重细节 光照效果（重要） 色差对比

图层

  • gold（内阴影，内发光）—内阴影突出字体凹效果，内发光加强效果质感

  • 五角星顶上高光

  • 五角星顶上亮部

  • 暗部（五角星暗部）

  • 五角星（斜面浮雕，描边，渐变叠加，外发光，投影）—加强周围的效果同步，显示出效果

  • 暗部（凹结构的投影暗部)

  • 金币凹结构（描边（内），描边（外），）—突出光照与凹效果，内描边体现凹效果，外描边加强光照效果

  • 暗部（底部半圆环投影暗部）

  • 金币顶层（描边。渐变叠加）—光照渐变效果

  • 金币渐变亮部—（顶端灯光照射）

  • 金币底（投影）

第九节 毛线质感思路及技法

  1. 制作毛线单个画笔素材
    （新建图层—圆角矩形—变形共形—调整素材—编辑—定义画笔预设）
  2. 钢笔工具—画好路径
    （路径—画笔描边路径）
  3. 毛边质感可以用画笔里的小草素材操作

第十节 图标的制作与规范

  1. 光线，阴影要注意
    （复制一层，调大羽化可做阴影和光照）
    （内阴影可用作高光，可不用斜面与浮雕）
  2. 颜色选取不要过于极端 右上（浅色）和左下（深色）

关于创作

  1. 确定主题
  2. 思考主题范围
  3. 选择元素，制作配色表
    （艺术来源于现实，多做参考）
